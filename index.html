Каскадные таблицы стилей

Сайты в интернете создаются при помощи HTML и CSS. HTML является основой страницы. Но страница созданная с помощью только языка HTML, не очень красива. HTML структурирует документ, CSS тесно взаимодействует с браузером, чтобы оформление сайта имело совершенный вид.

CSS используют для форматирования HTML-элементов, это форматирование включает в себя стиль - правило, описывающее внешний вид конкретной части вэб-страницы. Таблица стилей является набором таких стилей.

Правила CSS подключаются к сайту с помощью специальных тэгов в HTML в виде отдельных файлов или прописываются напрямую в структуре странички. CSS правила позволяют не только форматировать код HTML, но так же создавать анимацию на страничке, что делает её привлекательной и динамичной.

Время не стоит на месте, развивается Интернет, а так же и гаджеты, которыми мы пользуемся. Теперь каждый может при помощи мобильного телефона или планшета выйти в сеть. Это в свою очередь заставило разработчиков задуматься о том, что сайты должны быть привлекательными и работоспособными на любом устройстве. Теперь сайты должны быть отзывчивыми и адаптивными, что в свою очередь возможно реализовать с помощью современных возможностей CSS

Технологии CSS

Создатели CSS постоянно выпускают новые версии своих программных продуктов. Версия CSS1, появившаяся в 1996 году, стала основой для дальнейшего совершенствования языка каскадных таблиц стилей.В первой версии языка CSS были введены основные понятия и команды: базовая структура таблиц стилей, концепция селекторов и большая часть свойств CSS.

В версии CSS2 были добавлены новые возможности, включая совместимость с различными принтерами, мониторами и другими устройствами. Здесь также были добавлены новые селекторы и возможность точно позиционировать элементы на веб-страницах.

Flexbox

Ещё недавно макет для всех страниц HTML верстался с помощью таблиц, float и других свойств CSS, которые не очень хорошо подходят для стилизации сложных веб-страниц.

Flexbox — режим вёрстки, который был специально разработан для создания надёжных адаптивных страниц. Flexbox упростил правильное выравнивание элементов и их содержимого и теперь является предпочтительной системой CSS для большинства веб-разработчиков.

Grid

CSS Grid Layout система двумерного макета, оптимизированного для дизайна пользовательского интерфейса. Главная идея, лежащая в основе макета сетки, заключается в разделении веб-страницы на столбцы и строки. В образовавшиеся области сетки можно помещать элементы сетки, а управлять их размерами и расположением можно с помощью специальных свойств модуля.

Кроме того, благодаря своей способности явно размещать элементы в сетке, Grid Layout позволяет кардинально преобразовывать структуру визуального макета (отображаемого на экране), не требуя соответствующих изменений разметки.

Keyframes

CSS переходы и трансформации отлично подходят для создания визуального взаимодействия на основе изменений состояния. Чтобы лучше контролировать происходящее на экране, можно воспользоваться свойством CSS анимации для создания простой анимации на @keyframes.

CSS keyframes позволяет производить изменения постоянно и автоматически, а не только в ответ на события мыши, как в transition. С помощью кейфреймов можно менять CSS стили для заданного селектора в любой точке в момент смены состояния или события, определенного через JQuery (например, скролинг). Данное правило используется в паре со свойством animation, с помощью которого можно задать duration, timing function, delay и direction. Свойства типа transform, если они входят в анимацию, объявляются внутри правила @keyframes.
